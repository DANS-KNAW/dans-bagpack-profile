Welcome¶

This site documents contains the DANS BagPack Profile specifications. A DANS BagPack is a BagPack as specified by the Research Data Alliance (RDA) with extra requirements added by DANS.

  • v0.1.0 - CURRENT VERSION - simplified rules for the DANS BagPack Profile produced by the Data Stations and dd-vault-ingest.
  • v1.0.0 - FUTURE VERSION - robust rules for the DANS BagPack Profile produced by the Data Stations and dd-vault-ingest.
