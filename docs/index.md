Welcome
=======

This site contains the *DANS BagPack Profile* specifications. A DANS BagPack is a BagPack as specified by the Research Data Alliance (RDA) with extra
requirements added by DANS.

Current version:

* [DANS BagPack Profile v1.0.0](versions/1.0.0.md) (published on 2026-02-10)

Archived versions, see [here](versions/archived/index.md). 

The version numbers follow the [semantic versioning](https://semver.org/){:target=_blank} scheme:

> Given a version number `MAJOR.MINOR.PATCH`, increment the:
>
> * `MAJOR` version when you make incompatible API changes,
> * `MINOR` version when you add functionality in a backwards compatible manner, and
> * `PATCH` version when you make backwards compatible bug fixes.
